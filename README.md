# Description

- An application that takes data and displays it in a table.
- Created pagination by pages.
- Added sorting in ascending and alphabetical order.
- Changed number of page in URL

## For start

- npm install
- npm run start

- React
- RTK Query
- React icons
- Tailwind CSS
